## 글을 디비에 업로드 하는법
- html의 stcipt부분에서 editor.getHTML()으로 내용을 얻어낼 수 있다는 사실은 알고있었습니다. 
- 그런데 프론트에서 데이터를 보내는 곳과 분리되어있기 때문에 onsubmit으로 함수를 등록해주고
- 해당 함수로 들어갔을때 안보이는 form에 (작성한 글에 대한)데이터를 입력해두고 그 데이터를 전송하도록 하였습니다.
